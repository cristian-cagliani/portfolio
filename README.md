## 🚀 Descrizione del Progetto

Questo è il mio portfolio personale (versione 1.0), progettato per essere una vetrina professionale delle mie esperienze e competenze informatiche. Il sito include un'interfaccia dark mode moderna, animazioni dinamiche e un design responsivo, enfatizzando la mia passione per le reti informatiche e i sistemi cloud.

---

## 🛠 Tecnologie Utilizzate

Il progetto è stato sviluppato utilizzando tecnologie web standard, senza l'uso di framework complessi, per garantire le massime prestazioni, un basso tempo di caricamento e un alto livello di personalizzazione:

*   **Struttura:** HTML5 Semantico
*   **Stili:** CSS3 (Flexbox, CSS Grid, Custom Properties / Variabili)
*   **Logica e Animazioni:** Vanilla JavaScript
*   **Icone:** FontAwesome 6
*   **Tipografia:** Google Fonts (`Inter` e `JetBrains Mono`)

---

## ✨ Funzionalità Principali

-   **Animazione Interattiva nel Canvas:** Un'animazione "a ragnatela" nella sezione Hero che reagisce al movimento del mouse, per richiamare visivamente l'ambito del Networking.
-   **Terminale Geek Animato:** Un finto terminale bash (`root@ccagliani:~#`) inserito nella sezione "Chi Sono" che simula l'esecuzione di comandi di sistema (ping, script).
-   **UI / UX Moderna e Dark Mode:** Un tema scuro nativo con colori di accento vibranti per farsi notare e allo stesso tempo risultare professionale e riposante per la vista.
-   **Design 100% Responsivo:** L'interfaccia si adatta in maniera fluida a schermi desktop, tablet e smartphone, includendo un menu hamburger per dispositivi mobili.
-   **Effetti Scroll Reveal:** Animazioni fluide di comparsa progressiva degli elementi (fade-in e slide-up) mano a mano che l'utente scorre la pagina.

---

## 📂 Struttura del Sito

Il portfolio naviga verticalmente (Single Page Application style) ed è diviso in diverse sezioni fondamentali:

1.  **Header (Hero):** Presentazione iniziale d'impatto con titolo, qualifica e call-to-action per scendere nella pagina.
2.  **Chi Sono:** Breve biografia, focus sugli obiettivi di carriera (Network & Cloud Specialist) e vetrina visuale con la finestra del terminale bash.
3.  **Competenze (Skills):** Categorizzazione tematica delle mie abilità tecniche (Networking, Cloud & Sistemi, Programmazione & DB, Applicativi & Soft Skills).
4.  **Certificazioni:** Elenco e link per verificare le certificazioni professionali conseguite (AWS Cloud Practitioner, Microsoft Azure AZ-900, Microsoft Security SC-900).
5.  **Esperienza & Formazione:** Timeline visiva dei percorsi di studio (ITS Angelo Rizzoli, IISS A. Greppi) e delle esperienze lavorative/stage documentate.
6.  **Contatti:** Informazioni chiare per entrare in contatto, tramite email diretta o canali social (LinkedIn, GitHub).

## ⚙️ Come Eseguire il Progetto

Trattandosi di un sito puramente statico (frontend-only), è facilissimo eseguirlo in locale senza build steps particolari:

1.  **Clona il repository nel tuo computer locale:**
    ```bash
    git clone https://github.com/cristian-cagliani/portfolio.git
    cd portfolio
    ```
    *(In alternativa, puoi scaricare il file .zip dal repository)*

2.  **Apri il progetto:**
    *   Apri semplicemente con un doppio clic il file `index.html` all'interno del tuo browser web preferito (Chrome, Firefox, Safari, Edge).
    *   *Per sviluppatori:* Apri la cartella su Visual Studio Code e avvia **Live Server** per godere del ricaricamento automatico della pagina durante le modifiche.
